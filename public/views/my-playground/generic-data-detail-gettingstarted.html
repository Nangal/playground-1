Getting started

To help get you started there are some template projects, in the "Templates"" section, that you can use to get started with our APIs. You'll need to have Node installed, but once you do you'll be able to have a project running on your local machine in 5 easy steps. Simply:

  1. Clone or download a project from the "Templates" section
  2. Navigate to the new folder and run 'npm install'
  3. Add the config variable provided below to the resources/script.js file 
              var config={{connectionConfig}};
  4. Run 'node index'
  5. Open a browser and go to localhost:8000

Once you're up and running, you can now customize the project or start something new! Use the field explorer to get a better understanding of the data and determine the names of the fields to use in your project. If you create something cool with a Playground data set, contact us for submission into the Qlik Playground showcase!
