Getting started

To help get you started there are some template projects, in the "Templates" section, that you can use to get started with our APIs. You'll need to have Node installed, but once you do you'll be able to have a project running on your local machine in 5 easy steps.

Simply:

  1. Clone or download a project from the "Templates" section.
  2. Navigate to the new folder and run 'npm install' from the command line. This will install needed modules for the project.
  3. Edit the script.js file in the resources directory. Replace the empty config variable at the top with this: 
              var config={{connectionConfig}};
    This code contains your api keys and application names.
  4. Run 'node index' in the command line. You'll see this message: 
    listening on port 8000
    Your server is now running
  5. Open a browser and go to http://localhost:8000. The app is now running on your local machine.

Once you're up and running, you can now customize the project or start something new! Use the field explorer to get a better understanding of the data and determine the names of the fields to use in your project.
